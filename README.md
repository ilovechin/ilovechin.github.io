# 锦的创作资源导航（已可直接填内容）

你只需要编辑一个文件：`assets/content.js`

## 你要改的内容
- `brand`：页面标题与简介
- `links`：账号链接（小红书/视频号/抖音）
- `portfolio`：作品墙（标题、平台、简介、封面图、链接）
- `resources`：资源与模板
- `contacts`：联系方式
- `privateArea.password`：私密区密码
- `privateArea.items`：私密资料内容

## 发布
推送到 `main` 分支后，GitHub Pages 自动更新：
